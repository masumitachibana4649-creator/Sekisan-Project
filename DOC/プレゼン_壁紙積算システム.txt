プレゼン_壁紙積算アプリ 説明原稿
想定時間: 4分20秒（260秒）
発表条件: Zoomで5分以内
発表形式: 画面共有しながら、スライド順に説明する。

　　【1. 表紙】　　表示秒数: 18秒 / 積算秒数: 18秒
Python AI実践科の橘です。
これから修了作品として制作した「壁紙積算アプリ」について発表します。
これはPDF図面から、壁紙工事に必要な面積、ロール本数、金額を計算するWebアプリです。
ちなみに壁紙の積算とは、必要数量を算出し、見積もりにつなげる作業のことです。

　　【2. きっかけ】　　表示秒数: 28秒 / 積算秒数: 46秒
このアプリを作ったきっかけは、工務店を営む友人の、壁紙の積算作業の愚痴からでした。
話を聞くとPDF図面からの壁紙の積算は、印刷した図面から定規で壁の縦横をはかり、
電卓で計算する、手作業で手間がかかり計算ミスが起きやすい作業でした。
そこで、図面から自動計算し、人が確認して直せる形を目指しました。

　　【3. 概要】　　表示秒数: 16秒 / 積算秒数: 62秒
PDF図面を登録し、AIが部屋情報を読み取り、Pythonが計算し、結果表示とCSV出力する流れです。
部屋別の面積や、壁紙別のロール本数、金額をまとめて確認できます。

　　【4. AI】　　表示秒数: 17秒 / 積算秒数: 109秒
AIは、PDF図面から部屋名、壁面積、開口部面積、天井面積などを読み取る係です。
AIが読み取り、Pythonが計算し、人が確認する、という分担にしました。

　　【5. 計算式】　　表示秒数: 15秒 / 積算秒数: 92秒
壁面積は、周長かける天井高から開口部を引いて求めます。
必要面積にロス率を加え、1ロールで貼れる面積で割って、ロール本数を切り上げます。

　　【6. デプロイ】　　表示秒数: 12秒 / 積算秒数: 121秒
公開環境では、Render上でDjangoアプリを動かします。
データベースはPostgreSQL、PDF読取はAI APIを使い、秘密鍵やAPIキーは環境変数で管理します。

　　【7. 使い方】　　表示秒数: 7秒 / 積算秒数: 128秒
ここから、実際の操作の流れです。
ログイン、新規案件登録、結果確認、CSV出力の順に説明します。

　　【8. ログイン】　　表示秒数: 8秒 / 積算秒数: 136秒
ログインするとダッシュボードに移動します。
最近の積算と「積算を開始」ボタンから、新しい案件登録へ進みます。



　　【9. 新規案件登録 1】　　表示秒数: 7秒 / 積算秒数: 143秒
「積算を開始」を押して、新規案件登録へ進みます。
ここから案件情報と図面情報を入力します。

　　【10. 新規案件登録 2】　　表示秒数: 13秒 / 積算秒数: 156秒
案件名、顧客名、PDF図面を入力します。
使う壁紙と、AIに読ませる平面図・展開図のページも指定します。

　　【11. 計算処理】　　表示秒数: 15秒 / 積算秒数: 171秒
「計算する」を押すと、PDFの検証、AIへの読取依頼、部屋データの作成、Pythonでの集計が行われます。
処理中であることが分かる画面を表示します。

　　【12. 結果確認】　　表示秒数: 15秒 / 積算秒数: 186秒
計算結果では、必要面積、ロール本数、金額を確認できます。
壁紙別集計や部屋別明細も表示し、見積結果を画面上で確認できます。

　　【13. 編集ボタン】　　表示秒数: 7秒 / 積算秒数: 193秒
AIの読み取り結果は、必ず人が確認します。
必要があれば、編集ボタンから修正画面へ進みます。

　　【14. 編集画面】　　表示秒数: 13秒 / 積算秒数: 206秒
編集画面では、壁紙、面積、開口部、集計対象外の指定を修正できます。
AIの結果をそのまま使わず、人が直せる設計にしました。

　　【15. CSV出力ボタン】　　表示秒数: 7秒 / 積算秒数: 213秒
結果を確認したら、CSV出力ボタンを押します。
画面上の結果を、資料として使える形で出力します。

　　【16. CSV出力内容】　　表示秒数: 12秒 / 積算秒数: 225秒
CSVには、案件情報、正式見積金額、壁紙別集計、部屋別明細が入ります。
Excelで開いて、見積資料や確認資料として使えます。

　　【17. まとめ】　　表示秒数: 35秒 / 積算秒数: 260秒
まとめです。
今までのやり方では、図面を見ながら手作業で確認し、面積やロール本数を手計算し、結果を転記していました。
このアプリを使うと、PDF登録、AI読取、Python計算、CSV出力まで対応できます。
その結果、図面確認の時間短縮、計算ミスや転記作業の削減、見積結果の保存がしやすくなると考えています。
以上で発表を終わります。ご清聴ありがとうございました。
